## 🚦 Getting Started

### 1. Install the Magic Template
```bash
dotnet new install EFProjectWebApi.Template
```

### 2. Create Your Project (POOF! 🧙)
```bash
dotnet new EFProjectWebApi2 -n YourProjectName -o YourProjectDirectory
```

### 3. Run Like the Wind! 💨
```bash
cd YourProjectDirectory
dotnet run
```
